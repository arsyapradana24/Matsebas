Pengertian Fungsi Invers

Definisi

Misalkan fungsi f: A → B dengan f(a) = b. Fungsi invers dari f, dilambangkan dengan f⁻¹, adalah fungsi dari B ke A yang memenuhi f⁻¹(b) = a.

💡

Intuitif: Fungsi invers "membalik" pasangan berurutan. Jika f memetakan (x, y), maka f⁻¹ memetakan (y, x).

Notasi Penting

f⁻¹(x) ≠ 1/f(x)

Notasi f⁻¹ berarti invers fungsi, BUKAN 1 dibagi f(x)

Contoh Sederhana

Diberikan:

Fungsi f = {(1,2), (2,4), (3,6), (4,8)}

Maka:

Fungsi invers f⁻¹ = {(2,1), (4,2), (6,3), (8,4)}

✓

Perhatikan bahwa setiap pasangan (x,y) di f menjadi (y,x) di f⁻¹

Syarat Fungsi Memiliki Invers

Syarat utama: Suatu fungsi memiliki invers jika dan hanya jika fungsi tersebut bijektif (one-to-one dan onto).

1. Injektif (One-to-One)

Definisi: Fungsi f disebut injektif jika untuk setiap x₁, x₂ ∈ A:

f(x₁) = f(x₂) ⟹ x₁ = x₂

Artinya: Setiap elemen kodomain maksimal dipetakan oleh satu elemen domain.

2. Surjektif (Onto)

Definisi: Fungsi f: A → B disebut surjektif jika untuk setiap b ∈ B, ada a ∈ A sehingga f(a) = b.

Range(f) = Kodomain(f)

Artinya: Semua elemen kodomain terpetakan.

3. Bijektif (One-to-One dan Onto)

Fungsi f disebut bijektif jika f injektif sekaligus surjektif.

⚠️

Hanya fungsi bijektif yang memiliki invers!

Horizontal Line Test

Uji visual untuk injektif:

Fungsi f injektif jika dan hanya jika setiap garis horizontal memotong grafik f maksimal di satu titik.

✓ Injektif

Garis horizontal memotong 1 titik

✗ Tidak Injektif

Garis horizontal memotong >1 titik

Contoh: Fungsi Tanpa Invers

Fungsi: f(x) = x² untuk x ∈ ℝ

Mengapa tidak memiliki invers?

  • f(-2) = 4 dan f(2) = 4
  • Fungsi ini tidak injektif (tidak one-to-one)
  • Untuk f⁻¹(4), tidak jelas apakah hasilnya -2 atau 2
💡

Solusi: Batasi domain menjadi x ≥ 0, maka fungsi menjadi bijektif dan memiliki invers f⁻¹(x) = √x

Cara Menentukan Fungsi Invers

Berikut adalah langkah-langkah sistematis untuk menentukan fungsi invers:

  1. 1

    Tulis y = f(x)

    Nyatakan fungsi dalam bentuk persamaan dengan y sebagai variabel terikat.

    Contoh: f(x) = 2x + 3 → y = 2x + 3
  2. 2

    Tukar posisi x dan y

    Ubah semua x menjadi y dan semua y menjadi x.

    y = 2x + 3 → x = 2y + 3
  3. 3

    Selesaikan untuk y

    Isolasi variabel y di satu sisi persamaan.

    x = 2y + 3 → 2y = x − 3 → y = (x − 3)/2
  4. 4

    Ganti y dengan f⁻¹(x)

    Hasil y adalah fungsi invers yang dicari.

    f⁻¹(x) = (x − 3)/2
  5. 5

    Verifikasi (opsional)

    Periksa bahwa f(f⁻¹(x)) = x dan f⁻¹(f(x)) = x.

    f(f⁻¹(x)) = 2·((x−3)/2) + 3 = x − 3 + 3 = x ✓

Contoh Soal Bertahap

Contoh 1: Fungsi Linear

Soal: Tentukan invers dari f(x) = 3x + 6

Penyelesaian:

  1. Tulis y = 3x + 6
  2. Tukar x dan y: x = 3y + 6
  3. Selesaikan untuk y:
    3y = x − 6
    y = (x − 6)/3
  4. Jadi: f⁻¹(x) = (x − 6)/3

Verifikasi:

f(f⁻¹(x)) = 3·((x−6)/3) + 6 = x − 6 + 6 = x ✓

Contoh 2: Fungsi Pecahan

Soal: Tentukan invers dari f(x) = (2x + 1)/(x − 3), x ≠ 3

Penyelesaian:

  1. Tulis y = (2x + 1)/(x − 3)
  2. Tukar x dan y: x = (2y + 1)/(y − 3)
  3. Selesaikan untuk y:
    x(y − 3) = 2y + 1
    xy − 3x = 2y + 1
    xy − 2y = 3x + 1
    y(x − 2) = 3x + 1
    y = (3x + 1)/(x − 2)
  4. Jadi: f⁻¹(x) = (3x + 1)/(x − 2), x ≠ 2

Verifikasi:

f(f⁻¹(x)) = (2·((3x+1)/(x−2)) + 1)/((3x+1)/(x−2) − 3)
= ((6x+2+x−2)/(x−2))/((3x+1−3x+6)/(x−2)) = (7x/(x−2))/(7/(x−2)) = x ✓

Contoh 3: Fungsi Akar

Soal: Tentukan invers dari f(x) = √(x − 2) + 3, x ≥ 2

Penyelesaian:

  1. Tulis y = √(x − 2) + 3
  2. Tukar x dan y: x = √(y − 2) + 3
  3. Selesaikan untuk y:
    x − 3 = √(y − 2)
    (x − 3)² = y − 2
    y = (x − 3)² + 2
  4. Jadi: f⁻¹(x) = (x − 3)² + 2, x ≥ 3
💡

Catatan: Domain f⁻¹ adalah x ≥ 3 karena range dari f adalah y ≥ 3 (nilai minimum √(x−2) adalah 0 saat x=2, sehingga y minimum = 0+3 = 3).

Contoh 4: Fungsi Kuadrat (dengan pembatasan)

Soal: Tentukan invers dari f(x) = x², x ≥ 0

Penyelesaian:

  1. Tulis y = x² dengan x ≥ 0
  2. Tukar x dan y: x = y²
  3. Selesaikan untuk y:
    y² = x
    y = ±√x
  4. Karena domain awal x ≥ 0, maka kita ambil akar positif:
    y = √x
  5. Jadi: f⁻¹(x) = √x, x ≥ 0
⚠️

Penting: Tanpa pembatasan domain x ≥ 0, fungsi f(x) = x² tidak memiliki invers karena tidak injektif!

Kontrol Grafik

💡

Simetri terhadap y = x: Jika titik (a, b) ada pada grafik f(x), maka titik (b, a) ada pada grafik f⁻¹(x). Kedua titik ini selalu simetris terhadap garis y = x.

Sifat-Sifat Fungsi Invers

No Sifat Penjelasan
1 f(f⁻¹(x)) = x Komposisi fungsi dengan inversnya menghasilkan identitas
2 f⁻¹(f(x)) = x Komposisi invers dengan fungsi asli juga identitas
3 (f⁻¹)⁻¹ = f Invers dari invers adalah fungsi asli
4 Domain f⁻¹ = Range f Domain invers sama dengan range fungsi asli
5 Range f⁻¹ = Domain f Range invers sama dengan domain fungsi asli
6 (a,b) ∈ f ⟺ (b,a) ∈ f⁻¹ Titik pada grafik f dan f⁻¹ saling tertukar koordinatnya
7 Grafik f dan f⁻¹ simetris terhadap y = x Pencerminan terhadap garis y = x
8 Jika f naik, maka f⁻¹ naik Monoton naik dipertahankan
9 Jika f turun, maka f⁻¹ turun Monoton turun dipertahankan

Contoh Penerapan Sifat

Diketahui: f(x) = 3x − 5

Maka: f⁻¹(x) = (x + 5)/3

Verifikasi sifat 1:

f(f⁻¹(x)) = 3·((x+5)/3) − 5 = x + 5 − 5 = x ✓

Verifikasi sifat 2:

f⁻¹(f(x)) = ((3x−5)+5)/3 = 3x/3 = x ✓

Invers Fungsi Komposisi

Teorema Penting

Jika f dan g adalah fungsi-fungsi yang memiliki invers, maka:

(f ∘ g)⁻¹ = g⁻¹ ∘ f⁻¹

Perhatikan: Urutan dibalik!

Analogi: Sepatu dan Kaos Kaki

Bayangkan proses memakai kaos kaki (g) lalu sepatu (f):

  • Memakai: Kaos kaki dulu (g), kemudian sepatu (f) → (f ∘ g)
  • Melepas: Sepatu dulu (f⁻¹), kemudian kaos kaki (g⁻¹) → (g⁻¹ ∘ f⁻¹)
✓

Untuk "membatalkan" komposisi, kita harus membalik urutan operasinya!

Contoh Perhitungan

Diketahui:

  • f(x) = 2x + 1
  • g(x) = x³

Langkah 1: Cari (f ∘ g)(x)

(f ∘ g)(x) = f(g(x)) = f(x³) = 2x³ + 1

Langkah 2: Cari (f ∘ g)⁻¹(x)

Misalkan y = 2x³ + 1

Tukar: x = 2y³ + 1

Selesaikan: 2y³ = x − 1

y³ = (x − 1)/2

y = ∛((x − 1)/2)

(f ∘ g)⁻¹(x) = ∛((x − 1)/2)

Langkah 3: Verifikasi dengan g⁻¹ ∘ f⁻¹

f⁻¹(x) = (x − 1)/2

g⁻¹(x) = ∛x

(g⁻¹ ∘ f⁻¹)(x) = g⁻¹(f⁻¹(x)) = g⁻¹((x−1)/2) = ∛((x−1)/2)

✓

Terbukti: (f ∘ g)⁻¹ = g⁻¹ ∘ f⁻¹

Rangkuman

Poin-Poin Penting Fungsi Invers

  1. Definisi: Fungsi invers f⁻¹ membalik pasangan (x,y) menjadi (y,x). Jika f(a)=b, maka f⁻¹(b)=a.
  2. Syarat: Fungsi harus bijektif (injektif dan surjektif) untuk memiliki invers.
  3. Cara Mencari: (1) Tulis y = f(x), (2) Tukar x dan y, (3) Selesaikan untuk y, (4) Ganti y dengan f⁻¹(x).
  4. Sifat Utama: f(f⁻¹(x)) = x dan f⁻¹(f(x)) = x. Grafik f dan f⁻¹ simetris terhadap y = x.
  5. Komposisi: (f ∘ g)⁻¹ = g⁻¹ ∘ f⁻¹. Urutan dibalik!
  6. Domain & Range: Domain f⁻¹ = Range f, dan Range f⁻¹ = Domain f.
